# 🏎️ F1 Data Platform

An end-to-end Data Engineering project designed to collect, transform, store and analyze Formula 1 data using modern data engineering practices.

## 🎯 Objectives

* Build a production-inspired data platform.
* Create automated ETL pipelines.
* Apply the Medallion Architecture (Bronze, Silver and Gold).
* Orchestrate workflows using Apache Airflow.
* Store analytical data in PostgreSQL.
* Deliver business insights through Power BI dashboards.
* Follow software engineering best practices, including testing, documentation and CI/CD.

## 🛠️ Tech Stack

* Python
* PostgreSQL
* Apache Airflow
* Docker
* Power BI
* Git & GitHub
